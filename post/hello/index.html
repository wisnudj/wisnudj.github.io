Java Standard Class (Bag. 1)

Sat, Jan 2, 2021

Di bahasa pemrograman java ada class-class standard yang biasa digunakan.

String class

Di java string itu adalah object yang tentunya pasti punya representasi dari class nya. Method yang ada di kelas string itu banyak. Kalau tidak percaya bisa langsung cek di link berikut https://docs.oracle.com/en/java/javase/14/docs/api/java.base/java/lang/String.html.

Namun karena ini Dev Note (bukan death note) tentu saya harus harus bisa merangkum kira-kira method apa saja yang perlu saya ingat supaya sedikit-sedikit saya gak buka documentasi java.

Method Keterangan
String toLowerCase() Membuat String baru dengan format lower case
String toUpperCase() Membuat String baru dengan format upper case
int length() Mendapatkan panjang dari string
boolean startsWith(value) Mengetik apakah string dimulai dengan string value
boolean endsWith(value) Mengetik apakah string diakhiri dengan string value
String[] split(value) Memecah string menjadi arrray string 
String name = "Wisnu Dharma Jaya";
String lowerName = name.toLowerCase();							// result: "wisnu dharma jaya"
String upperName = name.toUpperCase();							// result: "WISNU DHARMA JAYA"
